# Dotfiles

> My dotfiles

---

This repository contains the configuration files I use on macOS or Linux computers.

## Getting started

Just clone the repository and copy to your home directory:

```bash
$ git clone git@github.com:tylucaskelley/dotfiles
$ cp dotfiles/src/* ~
```
